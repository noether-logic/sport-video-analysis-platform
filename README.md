# Платформа видеоразборов для спорта

Современная платформа для заказа профессиональных видеоразборов спортивных тренировок и игр.

## Особенности

- 🎥 Загрузка видео для анализа
- 👨‍🏫 Профессиональные разборы от тренеров
- 💳 Система оплаты через СБП
- 📊 Отслеживание статуса разборов
- 🎨 Современный Material Design 3 интерфейс
- 🏃 Поддержка различных видов спорта

## Технологии

- React 18
- TypeScript
- Tailwind CSS v4
- Vite
- Material Design 3

## Установка

```bash
pnpm install
```

## Запуск

```bash
pnpm dev
```

## Сборка

```bash
pnpm build
```
